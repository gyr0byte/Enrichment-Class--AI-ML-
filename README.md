# AI & ML Enrichment Class

This repository contains the notebooks, datasets, and practice material used for a college enrichment class focused on **AI, Machine Learning, and data analysis with Python**.

The content starts with core Python data analysis concepts, moves through NumPy and pandas, then into data cleaning and exploratory data analysis, and finishes with introductory machine learning and regression.

## Course Roadmap

### Week 1: Foundations

- Introduction to data analysis
- NumPy basics for numerical computation
- Introductory statistics for data analysis
- pandas fundamentals for tabular data

### Week 2: Data Wrangling and EDA

- Data cleaning techniques
- GroupBy and aggregation
- Exploratory Data Analysis (EDA)
- Working with real-world datasets
- Practice notebook using the Pokemon dataset

### Week 3: Machine Learning

- Introduction to machine learning and regression
- Linear regression concepts and practice
- Evaluation of regression models
- Training on a regression dataset

## Requirements

To work through the notebooks, you will typically need:

- Python 3.10 or newer
- Jupyter Notebook or VS Code with the Jupyter extension
- Common data science libraries such as:
  - `numpy`
  - `pandas`
  - `matplotlib`
  - `seaborn`
  - `scikit-learn`

## Quick Start

1. Clone or download the repository.
2. Open the folder in **VS Code** or **Jupyter Notebook**.
3. Open the notebook for the week you want to study.
4. Run the cells from top to bottom.
5. Use the CSV and Excel files in the same folder as the notebooks when a lesson refers to them.

## Recommended Workflow

- Start with `Week_01` if you are new to Python data analysis.
- Follow the notebook order inside each week, since later notebooks often build on earlier ones.
- Keep the datasets in place relative to the notebooks so file paths continue to work.
- Re-run notebooks after editing cells to make sure the outputs still match the lesson flow.
